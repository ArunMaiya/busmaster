Installation of Runtime Libraries

Description

Describes about the installation of additional libraries required for BUSMASTER compilation.

Support to IXXAT hardware

The following steps should be followed to compile CAN_IXXAT_VCI project in BUSMASTER.

1. Download and install IXXAT VCI drivers from the below link:


  • http://www.ixxat.com/download_vci_v3_en.html

2. Set Windows environment variable 'IXXAT_VCI_SDK' with the path to the VCI header files (e.g. IXXAT_VCI_SDK = "c:\Program Files\IXXAT\VCI 3.5\sdk\Microsoft_VisualC").

After the above two steps, please restart the machine.
